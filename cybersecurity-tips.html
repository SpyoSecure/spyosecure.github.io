Protect Yourself from Scammers

- Never share personal details or passwords online.
- Use multi-factor authentication (MFA) whenever possible.
- Verify emails and messages before clicking any links.
- If an offer sounds too good to be true, it probably is.
- Report scams to relevant authorities or cybersecurity teams.

What to Do If You Get a Virus

- Disconnect from the internet immediately.
- Run a full scan with trusted antivirus software.
- If it's ransomware, contact us to check if a decryption tool exists.
- Reset compromised passwords and enable MFA.
- If the infection is severe, consider reinstalling your OS.

How to Secure Your Wi-Fi Network

- Use a VPN that does not have logs, we recommend Mullvad VPN. Official website of Mullvad VPN.

- Change your default router login credentials.
- Use WPA3 or WPA2 encryption, never WEP.
- Disable WPS (Wi-Fi Protected Setup).
- Regularly update router firmware.
- Set up a guest network for visitors.

How to Set Up Quad9 DNS

- Open Network Settings on your Windows computer.
- Select "Change Adapter Options" and choose your active connection.
- Go to "IPv4 Settings" and set the DNS to **9.9.9.9** (Primary) and **149.112.112.112** (Secondary).
- Save the settings and restart your connection.

For a step-by-step guide, check out the official Quad9 tutorial on their YouTube channel: How to Set Up Quad9 DNS on Windows.

If you're using Linux, the setup process may vary depending on your distribution. It's best to look up the specific steps for your system to ensure proper configuration.
